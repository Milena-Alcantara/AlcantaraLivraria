
# Alcantara Livraria 📚

Bem-vindo à Alcantara Livraria! Um sistema de gerenciamento de livros, autores e usuários.

## Funcionalidades Principais

### Como Administrador 👑

- **Logar com um Token** 🔒: Para acessar as funcionalidades de administrador, faça login usando um token exclusivo.

- **Registrar um Livro 📖**: Adicione novos livros ao catálogo da livraria, incluindo título, autor, gênero e informações adicionais.

- **Registrar um Autor ✍️**: Adicione novos autores ao banco de dados, fornecendo detalhes como nome, biografia e outras informações relevantes.

- **Deletar um Livro 🗑️**: Remova livros do catálogo quando necessário, garantindo uma lista precisa de livros disponíveis.

- **Deletar um Autor 🗑️**: Remova autores que não são mais relevantes, observando se nenhum livro está associado a eles.

- **Deletar um Usuário 🚫**: Como administrador, você tem o poder de excluir usuários, garantindo um ambiente seguro.

- **Associar um Livro a um Autor 📚✍️**: Conecte livros a autores correspondentes, mantendo um registro preciso da autoria.

- **Associar um Autor a um Livro ✍️📚**: Também é possível associar autores a livros existentes, mantendo os detalhes do autor atualizados.

### Como Usuário 👤

- **Se Cadastrar 📝**: Usuários podem se registrar no sistema, fornecendo informações básicas e criando uma conta.

- **Fazer Login 🔑**: Após o registro, faça login para acessar as funcionalidades do usuário.

## Como Executar

Siga os seguintes passos para executar o sistema Alcantara Livraria:

1. Clone este repositório em sua máquina local.
2. Configure o ambiente e as dependências necessárias.
3. Inicie o sistema e escolha entre as opções de administrador ou usuário.

## Aviso

Este sistema é destinado apenas para fins de didáticos e não deve ser usado em produção sem medidas adicionais de segurança.
